# Historical Context Archive

This directory contains extracted historical context from various project AGENTS.md files and system logs, preserved for the memory skill.

## Files

- `paperclip_AGENTS.md` - AGENTS.md from the Paperclip project (control plane for AI-agent companies)
- `scottish-rite_AGENTS.md` - Deployment rules and user preferences for the Scottish Rite website
- `esamms_AGENTS.md` - Project notes and issue log for eSAMMs (electronic Southern Arizona Master Masons) Zo Site
- `HealthCheck.log` - Health check log from 2026-05-11 showing PostgreSQL down and service connectivity issues
- `system-health.log` - System status report from 2026-04-30 showing service recoveries and PostgreSQL restart

## Usage

These files can be searched using the `memory_search` tool to retrieve relevant historical instructions, decisions, and troubleshooting steps when working on related projects or encountering similar issues.

Example:
```
/memory_search "Scottish Rite deployment rule"
```
